Student Task Manager

 1. Description

This project is a simple command-line based task manager for students. It helps students keep track of their daily tasks and assignments. It also helps students when they have lot assignments to do.

 2. Features

* Add tasks
* View tasks
* Mark tasks as completed
* Delete tasks

3. Technologies Used

 Python

4. How to Run

a) Download the project
b) Open terminal
c) Run:
   python main.py

5. Usage

Follow the menu options to manage your tasks.

6. Purpose

This project was created as part of a BYOP (Bring Your Own Project) assignment to solve real-world student productivity problems.
